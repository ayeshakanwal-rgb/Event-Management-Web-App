# 🎉 Event Management Web App

A responsive and interactive Event Management Web Application built using HTML, CSS, and JavaScript. This project allows users to dynamically manage events by adding, deleting, searching, and viewing event details in real time.

- DOM Manipulation
- Flexbox & Grid Layout

---

## ✨ Features

- 📅 Display upcoming events dynamically
- ➕ Add new events using a form
- ❌ Delete events instantly
- 🔍 Search events by name or date
- ⚠️ Form validation (all fields required)
- ⏳ Automatic sorting of events by date
- 🎨 Past events highlighted differently
- 📱 Fully responsive design (mobile, tablet, desktop)
- 💡 Interactive UI with hover effects

## 🧠 How It Works

1. A predefined list of events is displayed on page load.
2. Users can add new events through the form.
3. JavaScript validates input before adding events.
4. Events are automatically sorted by date.
5. Users can delete events dynamically.
6. Search bar filters events in real-time.
